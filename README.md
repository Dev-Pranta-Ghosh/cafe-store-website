# ☕ Cafe Store – Responsive Coffee Shop Website

## Live Demo
https://caaffeestore.netlify.app/

## Project Overview
Cafe Store is a modern and responsive coffee shop website designed to showcase coffee products, menu items, and café information in a clean and engaging layout. The project focuses on responsive design, structured UI sections, and smooth user interaction to provide visitors with an enjoyable browsing experience.

This website can be used by cafés, coffee shops, or small food businesses to present their menu and highlight their products online.

## Features
- Fully Responsive Design
- Modern Landing Page Layout
- Coffee Menu Showcase
- Featured Products Section
- About Section
- Contact Form
- Smooth Hover Effects
- Clean UI Structure

## Technologies Used
HTML5  
CSS3  
JavaScript (Vanilla JS)

## Tools
Git  
GitHub  
Netlify (Deployment)

## Purpose of the Project
The purpose of this project was to practice building a responsive landing page while maintaining clean frontend code structure and modern UI design principles.
